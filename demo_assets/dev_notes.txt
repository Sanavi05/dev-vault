## Redis Configuration Issue (2024-03-10)
Redis connection kept timing out in production. Root cause: default 
connection pool size was 10, but we had 50 concurrent workers. 
Fix: set REDIS_POOL_SIZE=100 in .env and added connection retry logic.

## CORS Problem (2024-03-12)  
Frontend getting blocked on /api/upload. FastAPI CORS middleware was 
only allowing GET requests. Added "POST" to allow_methods in main.py.
Lesson: always explicitly list methods, don't rely on wildcard in staging.

## Database Migration Failure (2024-03-18)
Alembic migration failed on prod because the migrations table was locked 
by a long-running query. Fix: kill idle transactions first with:
SELECT pg_terminate_backend(pid) FROM pg_stat_activity WHERE state = 'idle';

## Async Bug in File Processor (2024-03-20)
FileProcessor.process() was calling a sync function inside an async route 
without run_in_executor. This was blocking the event loop and causing 
timeouts on large file uploads. Fix: wrap with asyncio.get_event_loop().run_in_executor(None, sync_fn)
